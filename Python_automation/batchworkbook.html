        # 代码实现思路

        实现批量创建工作簿并保存在指定目录中的步骤如下：

        1. 导入需要的库。我们需要os库来创建和检查目录，需要openpyxl库来创建Excel工作簿和工作表。
        2. 指定要保存工作簿的目录。这可以通过设置一个变量来实现。
        3. 检查目录是否存在。如果不存在，我们需要创建它。这可以通过os库的path.exists()函数来实现。如果目录不存在，os.makedirs()函数可以创建它。
        4. 循环创建工作簿。我们可以设置一个循环，例如for循环，来创建多个工作簿。每次循环都会创建一个新的工作簿。
        5. 创建新的工作簿。这可以通过openpyxl的Workbook()函数来实现。
        6. 保存工作簿到指定目录。我们可以使用openpyxl的save()函数来保存工作簿，使用os.path.join()函数来连接目录和文件名。

        这些步骤应该能够实现批量创建工作簿并保存在指定目录中的目标。

        # python代码实现

        import os  
from openpyxl import Workbook  
  
def create_workbooks(num_workbooks, output_folder):  
    for i in range(num_workbooks):  
        # 创建新的工作簿  
        wb = Workbook()  
  
        # 创建新的工作表并命名为 "第X个工作表"  
        ws = wb.active  
        ws.title = f"第{i+1}个工作表"  
  
        # 保存工作簿到指定文件夹  
        file_name = f"{output_folder}/工作簿{i+1}.xlsx"  
        wb.save(file_name)  
  
# 调用函数，创建100个工作簿  
create_workbooks(100, "output_folder")
